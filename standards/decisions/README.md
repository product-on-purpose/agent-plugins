# Standard decisions (ADRs)

Architecture Decision Records for the Advanced Skill Library Standard itself, in MADR format, numbered and immutable once accepted (Standard Section 10.4). Authored with the `askit-decision` skill (adr mode).

This trail is the **Standard's** governance record, distinct from any plugin's. It begins at 0001. The `agent-skills-toolkit` plugin's own ADRs (0020-0023) remain in that plugin and, where they shaped the Standard before governance was separated out (ADR 0001), stand as provenance rather than being renumbered here.
